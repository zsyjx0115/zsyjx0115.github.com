网络爬虫（又被称为网页蜘蛛，网络机器人，在FOAF社区中间，更经常的称为网页追逐者），是一种按照一定的规则，自动地抓取万维网信息的程序或者脚本。另外一些不常使用的名字还有蚂蚁、自动索引、模拟程序或者蠕虫。使用http模块实现小爬虫。

## HTTP小爬虫

	var http = require('http');
	var url = "";
	http.get(url, function(res){
		var html = '';
		
		res.on('data', function(data){
			html += data;
		});
		
		res.on('end', function(){
			console.log(html);
		});
	}).on('error', function(){
		console.log('error');
	});
